# Chatop-app - Backend

This project is the backend of the Chatop application,
a peer-to-peer rental platform.
It is developed in Java Spring Boot and uses JWT for authentication,
 and MySQL as the database.

## Prerequisites

- Java 11 or higher
- Maven 3.6 or higher
- MySQL (or another relational database)

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/AntoineGallou31/projet3.git
    cd projet3
    ```

2. Build the project with Maven:

    ```bash
    mvn clean install
    ```

3. Create a MySQL database for the project:

    ```sql
    CREATE DATABASE projet3;
    ```

    You will find the SQL scripts to create the tables in the `src/main/resources/data.sql` directory.

4. Modify the `application.properties` file to configure the database connection:

    ```properties
    spring.datasource.url=jdbc:mysql://localhost:5432/projet3
    spring.datasource.username=your_username
    spring.datasource.password=your_password
    spring.jpa.hibernate.ddl-auto=update
    ```

## Running the Project

1. Start the Spring Boot application:

    ```bash
    mvn spring-boot:run
    ```

2. The application will be accessible at: `http://localhost:3001`

## API Documentation

API documentation is automatically generated by Swagger and is accessible at: `http://localhost:3001/swagger-ui/index.html#/`
